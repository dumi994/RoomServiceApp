## Requisiti

- PHP >= 8.1
- Composer
- MySQL/MariaDB (o SQLite in locale)
- Node.js (>= 18) + npm
- Estensioni PHP comuni: `pdo`, `mbstring`, `openssl`, `tokenizer`, `xml`, `ctype`, `json`

## Installazione

Clona il repository ed entra nella cartella del progetto:

```bash
git clone <repo-url>
cd roomservice
```
